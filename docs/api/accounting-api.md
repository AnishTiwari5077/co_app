# SahakariMS — API: Accounting API

## Base URL
`/api/v1/accounting`

All endpoints require `Authorization: Bearer {token}`.

---

## Chart of Accounts

### GET /accounting/accounts
List all GL accounts.

**Query Parameters:**
| Param | Type | Description |
|-------|------|-------------|
| `type` | string | Asset \| Liability \| Equity \| Income \| Expense |
| `parentId` | uuid | Child accounts of parent |
| `search` | string | Code or name |
| `isActive` | bool | Filter active accounts |

**Response 200:**
```json
{
  "accounts": [
    {
      "id": "uuid",
      "accountCode": "1100",
      "accountName": "Cash and Cash Equivalents",
      "accountType": "Asset",
      "normalBalance": "Debit",
      "parentCode": "1000",
      "balance": 850000.00,
      "isActive": true,
      "level": 2
    }
  ]
}
```

---

### POST /accounting/accounts
Create new GL account.

**Permission:** `ACCOUNTING_COA_EDIT`

**Request:**
```json
{
  "accountCode": "4105",
  "accountName": "Processing Fee Income",
  "accountType": "Income",
  "parentId": "uuid",
  "description": "Loan processing fees collected"
}
```

---

### POST /accounting/vouchers
Create a manual journal voucher.

**Permission:** `ACCOUNTING_VOUCHER_CREATE`

**Request:**
```json
{
  "voucherType": "Journal",
  "voucherDateBs": "2081-04-15",
  "narration": "Correction entry — accrued interest",
  "entries": [
    {
      "accountId": "uuid",
      "debitAmount": 1500.00,
      "creditAmount": 0.00,
      "narration": "Interest expense"
    },
    {
      "accountId": "uuid",
      "debitAmount": 0.00,
      "creditAmount": 1500.00,
      "narration": "Interest payable"
    }
  ]
}
```

**Validation:**
- Sum of debits MUST equal sum of credits (balanced entry)
- Minimum 2 entries
- No entry can have both debit and credit non-zero

**Response 201:**
```json
{
  "id": "uuid",
  "voucherNumber": "JV-KTM-2081-04-00235",
  "status": "Draft",
  "totalDebit": 1500.00,
  "totalCredit": 1500.00
}
```

---

### POST /accounting/vouchers/{id}/post
Post a draft voucher to the ledger.

**Permission:** `ACCOUNTING_VOUCHER_POST`

**Response 200:**
```json
{ "status": "Posted", "postedAt": "2081-04-15T11:30:00Z", "postedBy": "Hari Accountant" }
```

---

### POST /accounting/vouchers/{id}/reverse
Reverse a posted voucher.

**Permission:** `ACCOUNTING_VOUCHER_REVERSE`

**Request:**
```json
{
  "reason": "Wrong account code used",
  "reversalDateBs": "2081-04-15"
}
```

**Response 200:**
```json
{
  "originalVoucherNumber": "JV-KTM-2081-04-00234",
  "reversalVoucherNumber": "JV-KTM-2081-04-00236",
  "status": "Reversed"
}
```

**Query Parameters:**

### GET /accounting/ledger/{accountId}
Account ledger — all transactions for a specific GL account.

**Query Parameters:** `fromDate`, `toDate` (BS format), `format`

---

### GET /accounting/fiscal-years
List fiscal years and their status.

### POST /accounting/fiscal-years
Create new fiscal year.

### POST /accounting/fiscal-years/{id}/close
**Permission:** `ACCOUNTING_YEAR_CLOSE`
Close the fiscal year. Locks all vouchers for that period and creates opening balances for next year.

## Cash Counter

### GET /accounting/cash-counter/sessions
List cashier sessions for today.

### POST /accounting/cash-counter/open
**Permission:** `CASH_OPEN`

```json
{
  "openingCash": 150000.00,

  "closingCash": 375000.00,
  "difference": 0.00,

}
```
