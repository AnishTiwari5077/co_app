# SahakariMS — Module: Asset Management

## Overview

The Asset module tracks cooperative-owned fixed assets — computers, furniture, vehicles, and buildings — for depreciation calculation, maintenance scheduling, and balance sheet reporting.

---

## Asset Categories

| Category | Depreciation Method | Useful Life |
|----------|-------------------|-------------|
| Buildings | Straight Line | 40 years |
| Furniture & Fixtures | Straight Line | 10 years |
| Computer & IT Equipment | Written Down Value (WDV) | 5 years |
| Vehicles | Written Down Value (WDV) | 8 years |
| Office Equipment | Straight Line | 7 years |

---

## Depreciation Methods

### Straight Line Method (SLM)
```
Annual Depreciation = (Cost - Salvage Value) / Useful Life

Example: Computer NPR 80,000, Salvage NPR 5,000, Life 5 years
  Annual = (80,000 - 5,000) / 5 = NPR 15,000/year
  Monthly = 15,000 / 12 = NPR 1,250/month
```

### Written Down Value Method (WDV)
```
Annual Depreciation = Book Value × Depreciation Rate

Example: Laptop NPR 80,000, Rate 20%
  Year 1: 80,000 × 20% = NPR 16,000  → Book Value: NPR 64,000
  Year 2: 64,000 × 20% = NPR 12,800  → Book Value: NPR 51,200
  Year 3: 51,200 × 20% = NPR 10,240  → Book Value: NPR 40,960
```

## Asset Data Model

```sql
CREATE TABLE assets (
    id                  UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    branch_id           UUID NOT NULL REFERENCES branches(id),
    asset_code          VARCHAR(30) NOT NULL UNIQUE,   -- AST-KTM-COMP-001
    asset_name          VARCHAR(200) NOT NULL,
    category            VARCHAR(50) NOT NULL,
    description         TEXT,
    purchase_date_ad    DATE NOT NULL,
    purchase_date_bs    VARCHAR(10) NOT NULL,
    cost_price          NUMERIC(18,4) NOT NULL,
    salvage_value       NUMERIC(18,4) NOT NULL DEFAULT 0,
    useful_life_years   INT NOT NULL,
    depreciation_method VARCHAR(20) NOT NULL,          -- StraightLine | WDV
    depreciation_rate   NUMERIC(5,2) NOT NULL,         -- Annual rate %
    accumulated_depreciation NUMERIC(18,4) NOT NULL DEFAULT 0,
    book_value          NUMERIC(18,4),                 -- Computed: cost - accumulated_dep
    vendor_name         VARCHAR(200),
    vendor_invoice      VARCHAR(100),
    location            VARCHAR(200),                  -- "Main Office — Room 2"
    assigned_to_user    UUID REFERENCES users(id),
    status              VARCHAR(20) NOT NULL DEFAULT 'Active',
    -- Status: Active | UnderRepair | Disposed | Transferred
    disposal_date_ad    DATE,
    disposal_amount     NUMERIC(18,4),
    disposal_reason     TEXT,
    is_deleted          BOOLEAN NOT NULL DEFAULT FALSE,
    created_at          TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at          TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE TABLE asset_depreciation_ledger (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    asset_id        UUID NOT NULL REFERENCES assets(id),
    fiscal_year     VARCHAR(10) NOT NULL,         -- "2081-82"
    period          VARCHAR(10) NOT NULL,         -- "2081-04" (YYYY-MM BS)
    opening_value   NUMERIC(18,4) NOT NULL,
    depreciation    NUMERIC(18,4) NOT NULL,
    closing_value   NUMERIC(18,4) NOT NULL,
    voucher_id      UUID REFERENCES accounting.vouchers(id),
    posted_at       TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

## Monthly Depreciation Job

```csharp
[DisableConcurrentExecution(300)]
public class MonthlyDepreciationJob
{
    public async Task ExecuteAsync(string fiscalPeriod)
    {
        var assets = await _assetRepo.GetActiveAssetsAsync();
        var voucher = Voucher.Create(VoucherType.Journal,
            DateOnly.FromDateTime(DateTime.Today),
            $"Monthly depreciation — {fiscalPeriod}",
            branchId: _branchId,
            createdBy: _systemUserId,
            isAutoGenerated: true);

        foreach (var asset in assets)
        {
            var monthlyDep = asset.DepreciationMethod == DepreciationMethod.StraightLine
                ? (asset.CostPrice - asset.SalvageValue) / (asset.UsefulLifeYears * 12)
                : asset.BookValue * (asset.DepreciationRate / 100 / 12);

            monthlyDep = Math.Round(monthlyDep, 2);

            if (monthlyDep <= 0 || asset.BookValue <= asset.SalvageValue) continue;

            // Accounting entry
            voucher.AddEntry(
                accountId: _depreciationExpenseAccountId,
                debit: monthlyDep, credit: 0,
                narration: $"Depreciation: {asset.AssetCode} — {asset.AssetName}");
            voucher.AddEntry(
                accountId: _accumulatedDepreciationAccountId,
                debit: 0, credit: monthlyDep,
                narration: $"Accumulated dep: {asset.AssetCode}");

            // Update asset record
            asset.AccumulatedDepreciation += monthlyDep;
            await _assetRepo.UpdateAsync(asset);
        }

        await _voucherService.PostAsync(voucher);
    }
}
```
